# MADGER Genesis Specification

## Network
- Blockchain: Solana
- Token standard: SPL Token
- Token name: MADGER
- Symbol: MADGER
- Decimals: 6
- Fixed supply: 1,000,000,000 MADGER
- Transfer tax: 0%

## Authorities
- Initial mint authority: Creator wallet
- Initial freeze authority: Creator wallet
- Mint authority will be revoked after verification and distribution
- Freeze authority will be revoked after verification

## Wallet Roles
- Creator: deployment and temporary authority management
- Treasury: long-term project reserves
- Operations: marketing, development, partnerships, and operating expenses
- Liquidity: initial and future liquidity provisioning
- Community: rewards, campaigns, airdrops, and ecosystem growth

## Proposed Allocation
- Liquidity: 60% — 600,000,000 MADGER
- Treasury: 20% — 200,000,000 MADGER
- Community: 10% — 100,000,000 MADGER
- Operations: 7% — 70,000,000 MADGER
- Creator reserve: 3% — 30,000,000 MADGER

## Safety Rules
- Never place seed phrases, private keys, or recovery codes in this repository
- Verify all wallet addresses before any transaction
- Rehearse deployment before mainnet
- Confirm metadata before revoking authorities
