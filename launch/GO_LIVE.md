# MADGER Launch Runbook

## Before Starting

- [ ] Verify all launch documentation is complete.
- [ ] Verify website and branding assets are finalized.
- [ ] Confirm the intended wallet addresses.
- [ ] Confirm the creator wallet has sufficient SOL for network fees.
- [ ] Review the token metadata and distribution plan.

## Launch Sequence

1. Create the token using the planned network parameters.
2. Verify the token information matches the genesis specification.
3. Confirm the fixed supply and intended allocations.
4. Establish initial liquidity according to the launch plan.
5. Verify the token is visible and tradable on the intended services.
6. Complete the planned authority changes only after all verification steps are finished.

## Public Announcement

- [ ] Publish the official token address.
- [ ] Publish launch announcement.
- [ ] Publish links to the website and documentation.

## Immediate Post-Launch

- [ ] Confirm token information is displayed correctly.
- [ ] Monitor network activity and community channels.
- [ ] Record any issues or follow-up actions.
